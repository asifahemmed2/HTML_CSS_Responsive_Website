# Loruki - Cloud Hosting Platform

A modern, responsive website for a cloud hosting platform. Loruki showcases cloud hosting services with a clean design, smooth animations, and a fully responsive layout.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Multi-page Structure**: Includes Home, Features, and Documentation pages
- **CSS Grid & Flexbox**: Utilizes modern CSS layout techniques for flexible and maintainable code
- **Font Awesome Icons**: Integration of Font Awesome for crisp, scalable icons
- **Custom Animations**: Smooth slide-in animations for enhanced user experience

## Pages

### Home (`index.html`)

- Hero section with demo request form
- Statistics showcase
- CLI demonstration
- Cloud hosting highlights
- Supported programming languages section

### Features (`features.html`)

- Platform overview
- Feature cards with icons

### Documentation (`docs.html`)

- Navigation sidebar
- Installation instructions
- CLI commands and requirements
- Getting started guide
